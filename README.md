# 🏛️ Repositório de Estágio — PROPEPG / DPG

Bem-vindo ao repositório central do meu estágio na **Diretoria de Pós-Graduação (DPG)**, vinculada à **Pró-Reitoria de Pesquisa e Pós-Graduação (PROPEPG)**. 

Este espaço é dedicado a centralizar todos os projetos, scripts e automações desenvolvidos durante meu período de atuação, servindo tanto como controle de versão corporativo quanto como portfólio de engenharia de software e análise de dados.

---

## 👨‍💻 Sobre o Meu Papel

Como estudante de Ciência da Computação na UFFS, meu objetivo principal neste estágio é aplicar conceitos sólidos de desenvolvimento e arquitetura de sistemas para modernizar, automatizar e otimizar os processos administrativos e acadêmicos da instituição. 

Minhas principais frentes de atuação incluem:
* **Automação de Rotinas Administrativas:** Redução do trabalho manual e mitigação de erros humanos através de scripts e robôs de processamento.
* **Engenharia de Dados e Integração:** Extração, transformação e carregamento (ETL) de dados a partir de documentos físicos e sistemas institucionais, utilizando Python, Regex e Inteligência Artificial.
* **Desenvolvimento de Ferramentas Internas:** Criação de microsserviços e esteiras de processamento que facilitam o dia a dia dos servidores da Pró-Reitoria.

## 📂 Portfólio de Projetos

Abaixo estão os módulos e soluções já integrados a este repositório. Cada diretório possui seu próprio `README.md` detalhado com arquitetura, tecnologias e instruções de uso.

### 1. [Automação de Diplomas](./Automacao_Diplomas)
Um ecossistema completo para a geração, auditoria e processamento em lote de diplomas de pós-graduação. O sistema extrai dados de processos via LLMs (Gemini/Groq), cruza as informações com o livro de registros oficial utilizando Regex para validação de segurança (Zero-Trust), e injeta os dados automaticamente em templates gerenciados pelo LibreOffice em background.

## 🛠️ Stack Tecnológico Base

As ferramentas e tecnologias mais frequentes em meus fluxos de trabalho aqui na DPG incluem:
* **Linguagens:** Python 3, Windows Batch Scripting.
* **Manipulação de Dados:** Pandas, Expressões Regulares (Regex).
* **Integrações e IA:** APIs REST, LLMs (Google Gemini, Groq), python-dotenv para segurança de credenciais.
* **Documentos:** Extração de metadados, processamento OCR e automação de LibreOffice (Headless CLI).
